# CodeAlpha_ExploratoryDataAnalysis-EDA

A Python-driven Exploratory Data Analysis project focused on Amazon product review data.

## Overview

This repository contains an EDA workflow implemented in `notebooks/EDA.ipynb`. The analysis uses the raw dataset at `data/raw/amazon.csv` and includes data inspection, summary statistics, missing-value review, and manual graph plotting.

## Highlights

- dataset: `data/raw/amazon.csv`
- notebook: `notebooks/EDA.ipynb`
- analysis steps: data loading, inspection, data types, summary statistics, missing-value checks, and manual visualization

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
